# Theme and Palettes Usage Example

This page provides various examples of using and customizing the UI theme and palettes.

<br/>

## Example - Default theme builder with core Neptune palette

This example shows how to change palette from default Unity to Neptune.

### Steps

Implement a custom ThemeProvider component that wraps DefaultThemeProvider and sets paletteName/paletteType parameters.

```jsx
    const ThemeProvider = (props) => <DefaultThemeProvider {...props} paletteName="neptune" paletteType="dark"/>;
```

<br/>

## Example - Default theme builder with custom palette

This example shows how to create and use a custom palette.

### Steps

Create the custom palette object.

```jsx
const customPalette = {
    displayName: "blue-grey/orange",
    palette: {
        type: "light",
        primary: {
            main: "#607D8B",
            contrastText: "#FFFFFF",
        },
        secondary: {
            main: "#FF9800",
            contrastText: '#212121'
        },
    }
};
```

Implement a custom ThemeProvider component that wraps DefaultThemeProvider and adds customPalette to the Palettes list. Also the custom palette name is passed as paletteName prop.

```jsx
    const ThemeProvider = (props) => <DefaultThemeProvider {...props} paletteName="customPalette" Palettes={{...Palettes, customPalette}}/>;
```

<br/>

## Example - Custom theme builder

This example shows how to create and use a custom theme builder to change/override UI styles.

### Steps

Implement function to produce [Material ThemeOptions](https://material-ui.com/customization/theming/#createmuitheme-options-args-theme)


```jsx
const buildTheme = (options, palette = Palettes.unity.palette) => ({
    typography: {
        fontSize: 12,
        body1: {
            fontSize: "12px"
        }
    },
    overrides: {
        MuiTableCell: {
            head: {
                fontSize: "11px",
            },
            root: {
                fontSize: "12px",
                whiteSpace: "pre-wrap",
            }
        }
        //put other theme options to override here...
    },
    options,
    palette,
});
```

Implement a custom theme builder by composing required [Material Theming API](https://material-ui.com/customization/theming)

```jsx
    const CustomThemeBuilder = (options, palette) => responsiveFontSizes(createMuiTheme(buildTheme(options, palette)));
```

Implement a custom ThemeProvider component that wraps DefaultThemeProvider and passes CustomThemeBuilder as Builder prop.

```jsx
    const ThemeProvider = (props) => <DefaultThemeProvider {...props} Builder={CustomThemeBuilder}/>;
```

<br/>

Custom ThemeProvider component created in these examples can be used as a container for any child components:

```jsx
<ThemeProvider>
    <FactoryContextProvider {...otherProps}>
        <Router>
            <PageContainer href={href}/>
        </Router>
    </FactoryContextProvider>
</ThemeProvider>
```
