# How to create Unity React App

## Prerequisites

1.	Download and install nodejs - https://nodejs.org/en/download/
2.	Setup Intellective NPM repository:
* Run the following command: npm set @intellective:registry http://18.236.47.14:4873
3.	Download and install create-react-app tool
* Run the following command: npm -g install yarn create-react-app

## Create Unity React App

1.	Run the following command to create unity react app (replace <app_name> with the actual app name):
create-react-app <app_name> --template @intellective/unity-origin

2.	Check out the README file in the project folder.
